# Project Report Outline

## Title
Student Mental Wellbeing Prediction Using Machine Learning

## Problem Statement
Students experience stress, anxiety, depression risk, and lifestyle imbalance. This project analyzes student lifestyle and mental health indicators to predict wellbeing risk and support early awareness.

## Objectives

- Study relationships between sleep, activity, academic performance, stress, mood, and depression risk.
- Build predictive models for mental health status and depression risk.
- Create a dashboard for analysis and user-friendly prediction.
- Provide an API for integration with other applications.

## Methodology

1. Collect and load student mental health and lifestyle datasets.
2. Clean missing values, duplicates, and target labels.
3. Encode categorical variables and scale numeric variables.
4. Train Random Forest classifiers.
5. Evaluate models using accuracy, weighted F1 score, and classification report.
6. Deploy results through Streamlit and Flask.

## Expected Outcome

A working ML-based student wellbeing project with trained models, visual analysis, prediction dashboard, and API endpoints.

## Limitations

The system is not a medical diagnosis tool. Predictions depend on dataset quality and should be used only for educational awareness.
